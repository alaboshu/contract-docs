# RWA Contracts API

欢迎查阅 RWA Contracts 的 API 文档。本项目旨在提供一套完整的智能合约接口，用于支持现实世界资产（Real World Assets, RWA）的链上管理与交互。

## 项目概述

RWA Contracts 提供了多个模块化的接口，涵盖 AMM、桥接、衍生品、设备管理、治理、KYC、MPC、预言机、RSY 以及零知识证明等多个领域。这些接口支持构建去中心化金融（DeFi）应用，实现资产发行、交易、质押、治理等功能。

## 主要模块

- **AMM**：提供流动性管理、交易对创建与交易路由等功能。
- **桥接（Bridge）**：支持跨链资产注册、封装与转移。
- **衍生品（Derivatives）**：支持期权创建与收益衍生品的行权与结算。
- **设备（Device）**：支持 NFT 铸造、销毁与委托管理。
- **治理（Governance）**：支持基于 NFT 的投票与 DAO 治理机制。
- **KYC**：提供合规性验证与用户白名单管理。
- **MPC**：支持多方计算钱包的创建、恢复与执行。
- **预言机（Oracle）**：提供跨链价格同步与多源价格聚合。
- **RSY**：支持法币存取与链上资产监控。
- **零知识证明（ZKP）**：支持数据锚定与证明验证。
